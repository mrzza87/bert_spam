# 📰 News Sentiment API

🚀 **A lightweight Flask API for sentiment classification on news headlines using CountVectorizer + Logistic Regression.**

![Python](https://img.shields.io/badge/python-3.10+-blue)
![Flask](https://img.shields.io/badge/Flask-API-green)
![Deploy](https://img.shields.io/badge/Deploy-Render-purple)

---

## 🌟 Features
✅ Classifies news headlines into sentiment categories  
✅ REST API with `/predict` endpoint  
✅ `/test` route with a simple HTML form  
✅ Easy deployment to **Render / Railway / Docker**

---

## 🏃 Quickstart (Local)

```bash
git clone https://github.com/YOUR_USER/news-sentiment-api.git
cd news-sentiment-api
